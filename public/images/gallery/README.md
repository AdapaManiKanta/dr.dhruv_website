# 📸 GALLERY PHOTOS — Drop your clinic photos here!

## How to add gallery photos:

Save your clinic photos in THIS folder (`public/images/gallery/`) with these names:

| File Name           | Description                          |
|---------------------|--------------------------------------|
| `gallery-1.jpg`     | Main large photo (shows biggest)     |
| `gallery-2.jpg`     | Home rehabilitation photo            |
| `gallery-3.jpg`     | Exercise therapy photo               |
| `gallery-4.jpg`     | Patient care photo                   |
| `gallery-5.jpg`     | Any clinic/session photo             |

## Photo tips:
- Landscape photos work best (16:9 or 4:3)
- Good lighting is key — avoid dark or blurry images
- Show real home visits, patient sessions, equipment
- JPG or PNG, keep under 800KB per photo

## What happens automatically:
- If file EXISTS → shows your real photo
- If NOT added → shows a stock Unsplash fallback image
