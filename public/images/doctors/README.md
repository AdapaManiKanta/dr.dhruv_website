# 📸 DOCTOR PHOTOS — Drop your files here!

## How to add doctor photos:

1. Save each doctor's photo in THIS folder (`public/images/doctors/`)
2. Name the files EXACTLY as shown below:
   - `dr-dhruva.jpg`       ← Dr. Dhruva (Main Lead)
   - `dr-raghavendra.jpg`  ← Dr. Raghavendra
   - `dr-prathyusha.jpg`   ← Dr. Prathyusha
   - `dr-mahesh.jpg`       ← Dr. Mahesh

## Photo tips for best look:
- Use a square or portrait crop (e.g. 400×400 or 400×500 px)
- Good lighting, professional background preferred
- White/light or clinic background works best
- JPG or PNG format, keep under 500KB per photo

## What happens automatically:
- If the photo file EXISTS → it shows the real photo
- If NOT yet added → shows the colored initials avatar (as a fallback)
